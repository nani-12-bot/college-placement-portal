# 🎓 College Placement Portal

A web-based placement management system that connects students, companies, and administrators.

## Features

### Student Module
- Student Registration
- Student Login
- Apply for Jobs
- Track Applications

### Company Module
- Company Registration
- Company Login
- Post Job Openings
- View Applicants

### Admin Module
- Approve Companies
- Manage Jobs
- Monitor Placements

## Technologies Used

- HTML5
- CSS3
- Python (Flask)
- MySQL
